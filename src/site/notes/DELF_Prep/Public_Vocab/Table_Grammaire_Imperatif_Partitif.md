# 📝 Vocabulaire : L'Impératif et le Partitif

| Français [Phonétique] | Anglais / Usage | Note |
| :--- | :--- | :--- |
| Parle ! [paʁl] | Speak! (tu) | No 's' for -ER verbs. |
| Finis ! [fini] | Finish! (tu) | |
| Parlons ! [paʁlɔ̃] | Let's speak! (nous) | |
| Parlez ! [paʁle] | Speak! (vous) | Plural or formal. |
| Sois / Soyons / Soyez [swa / swajɔ̃ / swaje] | Be! | Impératif d'Être. |
| Aie / Ayons / Ayez [ɛ / ɛjɔ̃ / ɛje] | Have! | Impératif d'Avoir. |
| Sache / Sachons / Sachez [saʃ / saʃɔ̃ / saʃe] | Know! | Impératif de Savoir. |
| Veuillez [vœje] | Please / Kindly | Polite request (Vouloir). |
| Lève-toi ! [lɛv twa] | Get up! | Reflexive positive. |
| Ne te lève pas ! [nə tə lɛv pa] | Don't get up! | Reflexive negative. |
| Vas-y ! [vazi] | Go ahead! / Go there! | 's' returns before 'y'. |
| du [dy] | some (M) | de + le |
| de la [də la] | some (F) | |
| de l' [də l] | some (vowel/h) | |
| des [de] | some (plural) | de + les |
| ne... pas de [nə pa də] | no / not any | Negation rule. |
| beaucoup de [boku də] | a lot of | Quantity rule. |
| un peu de [œ̃ pø də] | a little bit of | |
| trop de [tʁo də] | too much of | |
| assez de [ase də] | enough of | |
| faire du sport [fɛʁ dy spɔʁ] | to do/play sports | |
| jouer de la musique [ʒwe də la myzik] | to play music | |
